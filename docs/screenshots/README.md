# Screenshots

This folder contains demo screenshots of the application.

## Available Screenshots:

1. **streamlit_main.png** - Main Streamlit interface
2. **prediction_results.png** - Churn prediction results with gauge
3. **batch_prediction.png** - Batch prediction interface
4. **api_docs.png** - Swagger UI API documentation

## How to Add Screenshots:

1. Run the application
2. Take screenshots of key features
3. Save them in this folder with descriptive names
4. Update README.md with actual screenshot paths

## Screenshot Guidelines:

- Use PNG format for better quality
- Recommended size: 1200x800 pixels
- Show actual functionality, not mockups
- Include UI elements clearly visible
- Crop unnecessary parts
